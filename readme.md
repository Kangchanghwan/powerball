

## 파워볼 시뮬레이터 (2012년 이후 정책)

- 추첨 방식은 로또와 비슷
- 파워플레이 제외
- 1 ~ 69 사이의 숫자 5개 / 1 ~ 26 사이의 숫자 1개를 맞추어야 1등
- 장당 2달라

### 배당표

등수 / 맞춘 개수 / 당첨금

- 1등 : 5개 + 1개, 누적 모두
- 2등 : 5개 , 100만 달러
- 3등 : 4개 + 1개, 5만달러 
- 4등 : 4개, 100달러
- 5등 : 3개 + 1개 , 100달러
- 6등 : 3개 , 7달러
- 7등 : 2개 + 1개 : 7달러
- 8등 : 1개 + 1개, 4달러
- 0등 : 0개 + 1개, 4달러

### 기능

## v1

 - 사용자는 프로그램 종료시까지 지속적으로 몇 개의 파워볼을 구매할 것인지 입력할 수 있다.
 - 사용자가 N (> 0) 을 입력하면 해당 개수만큼 파워볼을 자동 구매하며
   - 당첨번호가 출력 및 당첨 등수가 나오며  손익이 계산된다. 
 - 사용자가 0을 입력시 프로그램은 종료된다.
